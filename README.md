# Sterowanie Procesów / Process Control

Semestr 4 (Zimowy 2024/2025) / Semester 4 (Winter 2024/2025)

## Opis przedmiotu

W ramach zajęć implementowano, testowano i optymalizowano algorytmy sterowania w MATLAB. Przeprowadzono eksperymenty związane z regulacją PID, DMC oraz ze sprzężeniem od stanu. Analizowano odpowiedzi skokowe, dobierano parametry regulatorów i porównywano różne metody sterowania. W repozytorium znajdują się sprawozdania i projekty związane z przedmiotem **Sterowanie Procesów**.

## Lista projektów
1. **Projekt 1 - Regulacja procesów ciągłych**
   Analiza otrzymanego modelu ciągłego, implementacja regulatora ze sprzężeniem od stanu oraz obserwatora pełnego rzędu.

2. **Projekt 2 - Regulacja procesów dyskretnych**
   Analiza otrzymanego modelu oraz dyskretyzacja. Implementacja i porównanie regulatorów PID, DMC oraz GPC.

---

## Course Description

During the course, control algorithms were implemented, tested, and optimized in MATLAB. Experiments were conducted with PID, DMC, and state-space control algorithms. Step responses were analyzed, controller parameters were selected, and different control methods were compared. The repository contains reports and projects related to the **Process Control** course.

## List of Projects
1. **Project 1 - Continuous Process Control**
   Analysis of the given continuous model, implementation of a state-space controller and a full-order observer.

2. **Project 2 - Discrete Process Control**
   Analysis of the given model and discretization. Implementation and comparison of PID, DMC, and GPC controllers.
